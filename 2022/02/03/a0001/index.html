# 環境

実行環境 用途
Ubuntu 20.04 初期構築
Windows 11 & macOS 12 ブログ更新

全てWindows、macOSで実施できるものの、この時は妻に借りたPCで作業していたので、Nodeとかインストールするのもなぁと思い、WebARENA Indigoに一時的にUbuntuサーバ借りて、VuePressの初期構築だけやりました。
作りきってからの更新時のビルドはGithub Actions上で行われるのでGitだけあれば更新できる環境になったはず。
テーマとか設定とかをいじりたい人はGUIでブラウザ環境が使えるクライアントOSのみで実施することをお勧めします。

# 概要図(ブログ更新)

# Public リポジトリにGithub Pagesを開設

Githubのリポジトリに「username/username.github.io」というPublicリポジトリを作ると、ホスティングサービスであるGithub Pagesが作れます。

# Private リポジトリにブログの更新環境を作る

Publicリポジトリにビルド前のVuePressコードが載るのが嫌だったので、ソースはPrivateリポジトリで作成して、ビルドしたアウトプットのみをPublicリポジトリにデプロイすることにしました。 「username/blog」というPrivatreリポジトリを作ります。

# VuePressでブログを作る

今回はブログ環境として静的サイトジェネレータのVuePressを利用します。
またブログ用途で利用するので、「@vuepress/plugin-blog」「@vuepress/theme-blog」のプラグインも使用します。

#

# Indigo上のUbuntu 20.04で下記を実施

## Nodeインストール
$ sudo apt install nodejs npm

## Yarn Classicインストール
$ sudo npm install --global yarn

## Git Cloneでブログ用リポジトリを取得
$ git config --global user.email "メールアドレス"
$ git config --global user.name "ユーザー名"
$ git clone https://github.com/ユーザー名/blog.git
$ cd blog/

## Manual インストール※Quickインストールだと一階層深くて自動デプロイがうまく動かなかった
$ yarn create vuepress-site
$ yarn init
$ yarn add -D vuepress
$ mkdir docs && echo '# Hello VuePress' > docs/README.md
$ cd docs/
$ yarn install
$ yarn docs:build
$ vi package.json

## 下記を追記
{
  "scripts": {
    "docs:dev": "vuepress dev docs",
    "docs:build": "vuepress build docs"
  }
}

# Github Actionsの有効化

$ cd
$ cd blog/docs
$ mkdir -p .guthub/workflows
$ cd .guthub/workflows
## 自動化アクションを下記に記載
$ vi vuepress-deploy.yml

name: CI
on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]

  workflow_dispatch:

jobs:
  # Build and Deploy
  build-and-deploy:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v2
        env: 
          TZ: "Asia/Tokyo"

      - name: vuepress-deploy
        uses: jenkey2011/vuepress-deploy@master
        env:
          TZ: "Asia/Tokyo"
          ACCESS_TOKEN: ${{ secrets.ACCESS_TOKEN }}
          TARGET_REPO: ユーザー名/ユーザー名.github.io
          TARGET_BRANCH: main
          BUILD_SCRIPT: yarn && yarn docs:build
          BUILD_DIR: docs/.vuepress/dist
